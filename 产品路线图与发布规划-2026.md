### 3.2 Tagline 候选（60字符内）

```
AI roundtable: multiple experts debate any question in depth
Multi-agent AI debate simulator for exploring complex topics  
Assemble an AI expert panel to debate any question
```

### 3.3 截图拍摄顺序建议
1. 首页空白状态（展示界面设计感）
2. 输入话题后 AI 确认 + 圆桌配置界面
3. 讨论进行中（多角色发言，展示不同观点）
4. 知识库上传 + 文件说明界面
5. 网页证据链结果展示

### 3.4 描述文案（草稿，供参考和修改）

```
Roundtable Braintrust is an open-source AI-powered discussion simulator 
that assembles a panel of specialized AI experts to debate any question 
from multiple perspectives.

Key features:
• 6 customizable AI seats — each with a unique persona, expertise, and stance
• Evidence chain — AI agents search the web (DuckDuckGo, Wikipedia) and cite sources in real time
• Knowledge base — upload your own documents (PDF, Word, TXT) and the AI references them during debate
• Works with any OpenAI-compatible API (OpenAI, Anthropic, local models via Ollama…)
• 100% browser-based — no server, no account, your data stays local

Perfect for: researchers, writers, consultants, students exploring complex decisions.
```

### 3.5 不需要安装包！
Product Hunt 上发布 **Web 应用直接给 Demo URL 就够了**。  
对于想本地部署的用户，README 里已经有一键启动说明（`serve-static.ps1`）。  
桌面安装包（.exe）是可选的增值项，**不影响 PH 发布**。

## 四、中期功能路线图（1-3个月）

### 4.1 知识库增强
- [ ] **相邻段落上下文**：检索命中片段时，同时取前后各一段，减少"断章取义"
- [ ] **文件摘要索引**：大文件先生成全文摘要，检索时优先比对摘要再深入
- [ ] **跨文件关联**：同一话题的多文件内容融合展示

### 4.2 讨论体验改进
- [ ] **讨论记录导出**：支持 Markdown / JSON 格式，方便分享或归档
- [ ] **讨论模板市场**：预设角色组合（法律团队、投资委员会、产品评审……）一键导入
- [ ] **角色记忆**：保存历史讨论中使用的自定义角色，下次直接复用
- [ ] **移动端适配**：CSS 响应式，支持手机和平板

### 4.3 协作与分享
- [ ] **讨论链接分享**：生成只读分享链接（需要后端或第三方存储）
- [ ] **截图/分享卡片**：一键生成美观的讨论摘要图片

## 五、长期路线（3个月以上）

### 5.1 后端服务（解锁多设备、协作）
- 轻量后端（Python FastAPI 或 Vercel Edge Functions）
- 知识库云端存储（用户可跨设备访问）
- 多人实时协作讨论（WebSocket）

### 5.2 商业化方向探索
- **免费版**：有限 API 调用、本地知识库
- **专业版**：云端知识库、更多角色席位、团队空间
- **企业版**：私有部署、自定义角色体系、API 接入

### 5.3 社区生态
- 角色/话题模板社区（GitHub 贡献）
- 多语言支持（目前已有中/英，可扩展日/韩/西）
